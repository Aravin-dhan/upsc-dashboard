üõ†Ô∏èStep-by-Step Instructions

1Prepare Your Code

First, we need to get your code ready for the big move!

This is like packing your belongings before moving to a new house. We want to make sure everything is organized and ready to go!

What to do:

  1. Open your computer's terminal (the black window with text)
  2. Navigate to your UPSC Dashboard folder
  3. Run this magic command:
./scripts/deploy-production.sh
This script will automatically prepare everything for you! It's like having a helpful robot pack your suitcase.

2Create a GitHub Account

GitHub is like a safe storage place for your code in the cloud.

Think of GitHub as a super secure bank vault where you store your precious code instead of money. It's free and millions of developers use it!

What to do:

  1. Go to github.com
  2. Click "Sign up" (it's free!)
  3. Choose a username (like your online nickname)
  4. Use your email and create a password
  5. Verify your email when GitHub sends you a message

3Upload Your Code to GitHub

Now we'll put your code in that safe storage place we just created.

This is like taking photos of your important documents and storing them in a cloud service like Google Drive, but for code!

What to do:

  1. In GitHub, click the green "New" button
  2. Name your repository "upsc-dashboard" (or any name you like)
  3. Make it "Public" so Vercel can see it
  4. Follow GitHub's instructions to upload your code
Don't worry if this seems confusing! GitHub has helpful guides, and there are lots of YouTube videos that show exactly how to do this.

4Create a Vercel Account

Vercel is the company that will host your website and make it available to everyone on the internet.

Vercel is like a landlord who gives you a prime location on the internet's busiest street to display your website. And the best part? It's completely free!

What to do:

  1. Go to vercel.com
  2. Click "Sign up"
  3. Choose "Continue with GitHub" (this connects your accounts)
  4. Allow Vercel to access your GitHub account

5Deploy Your Website

This is the exciting part! We're going to make your website live!

This is like cutting the ribbon at a grand opening. Your website will go from being a private project to a public website that anyone can visit!

What to do:

  1. In Vercel, click "New Project"
  2. Find your "upsc-dashboard" repository and click "Import"
  3. Vercel will automatically detect it's a Next.js project (smart!)
  4. Click "Deploy" and wait for the magic to happen
Vercel will now build your website and give you a free URL like "your-project.vercel.app"

üîßEnvironment Variables (The Secret Settings)

Your website needs some secret settings to work properly. Think of these like the keys to your house!

Environment variables are like secret passwords that your website needs to function. Just like you need a key to unlock your front door, your website needs these special keys to unlock its features.

What to do:

  1. In your Vercel project, go to "Settings" ‚Üí "Environment Variables"
  2. Add these important variables:
NEXTAUTH_SECRET=your-super-secret-key-here NEXTAUTH_URL=https://your-project.vercel.app ADMIN_EMAIL=your-email@example.com ADMIN_PASSWORD=your-secure-password
Make sure to replace "your-project" with your actual Vercel project name, and use your real email and a strong password!

üÜòTroubleshooting Common Problems

Don't panic if something doesn't work! Here are solutions to common issues:

‚ùåProblem: "Build Failed" Error

What this means: Vercel couldn't build your website (like a recipe that didn't work)

Solution:

  1. Check the build logs in Vercel for error messages
  2. Make sure all your files are uploaded to GitHub
  3. Try running npm run build on your computer first

üîêProblem: Can't Log In

What this means: The login system isn't working properly

Solution:

  1. Double-check your environment variables in Vercel
  2. Make sure NEXTAUTH_URL matches your actual website URL
  3. Verify your admin email and password are correct

üêåProblem: Website Loads Slowly

What this means: Your website is taking too long to load

Solution:

  1. Wait a few minutes - first visits are sometimes slower
  2. Check your internet connection
  3. Try visiting from a different device or location

üì±Problem: Doesn't Work on Mobile

What this means: The website doesn't display properly on phones

Solution:

  1. Clear your mobile browser cache
  2. Try a different mobile browser
  3. Check if the issue exists on desktop too
If you're still having problems, don't worry! You can always ask for help on GitHub, Stack Overflow, or the Vercel community forums. The developer community is very friendly and helpful! ü§ù
